# சர்வம லோக்கலைஸ்

சர்வம AI மொழிபெயர்ப்பு மூலம் இயங்கும் வெப்சைட் லோக்கலைசேஷன் CLI.

`sarvam-localize` உங்கள் JSON லோக்கல் மற்றும் மார்க் டவுன் கோப்புகளை இந்திய மொழிகளில் மொழிபெயர்க்க உதவுகிறது, அதே நேரத்தில் உள்ளமைக்கப்பட்ட சாவி அமைப்பு மற்றும் மாறிகளைப் பாதுகாக்கிறது.

## தேவைகள்

உங்கள் சாவியை இதிலிருந்து பெறுங்கள்: https://dashboard.sarvam.ai.

மற்றும் அதை `.env` கோப்பில் ஒட்டவும்

```bash
SARVAM_API_KEY=your_api_key
```

## நிறுவுதல்

நிறுவாமல் இயக்க:

```bash
npx sarvam-localize
```
