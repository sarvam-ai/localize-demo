# Sarvam Localize

വെബ്സൈറ്റ് ലോക്കലൈസേഷൻ CLI സർവം AI ട്രാൻസ്ലേഷൻ വഴി പ്രവർത്തിക്കുന്നു.

`sarvam-localize` നിങ്ങളെ ഇന്ത്യൻ ഭാഷകളിലുടനീളം JSON ലോക്കേൽ & മാർക്ക്ഡൗൺ ഫയലുകൾ വിവർത്തനം ചെയ്യാൻ സഹായിക്കുന്നു, അതേസമയം നെസ്റ്റഡ് കീ ഘടനയും വേരിയബിളുകളും സംരക്ഷിക്കുന്നു.

## ആവശ്യകതകൾ

നിങ്ങളുടെ കീ ഇവിടെ നിന്ന് നേടുക: https://dashboard.sarvam.ai.

ഇത് `.env` ഫയലിൽ പേസ്റ്റ് ചെയ്യുക

```bash
SARVAM_API_KEY=your_api_key
```

## ഇൻസ്റ്റാൾ ചെയ്യുക

ഇൻസ്റ്റാളേഷൻ ഇല്ലാതെ പ്രവർത്തിപ്പിക്കുക:

```bash
npx sarvam-localize
```
